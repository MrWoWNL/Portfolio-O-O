# ✅ Quick Start Checklist

Welcome to your new portfolio template! Follow this checklist to get your portfolio live in 20 minutes.

---

## 📋 Setup Checklist

### ☐ Step 1: Install Dependencies (5 min)
```bash
cd portfolio-template
npm install
npm run dev
```
Visit: http://localhost:5173

---

### ☐ Step 2: Edit Your Information (10 min)

Open: `src/data/portfolioData.js`

#### Personal Info
- [ ] Change `name` to your full name
- [ ] Update `role` (e.g., "Frontend Developer")
- [ ] Write your `tagline`
- [ ] Add your `bio` (2-3 sentences)
- [ ] Set your `location`

#### Education
- [ ] Update degree name
- [ ] Change institution
- [ ] Update period (years)
- [ ] Add your GPA/achievements

#### Skills
- [ ] Remove skills you don't have
- [ ] Add your actual skills
- [ ] Check icon names at react-icons.github.io

#### Projects
- [ ] Replace example projects with yours
- [ ] Add project descriptions
- [ ] Update tech stacks
- [ ] Add GitHub links
- [ ] Add live demo links (if available)
- [ ] Mark 2-3 as `featured: true`

#### Experience
- [ ] Add internships
- [ ] Add training/bootcamps
- [ ] Add hackathon wins
- [ ] Add achievements
- [ ] Set correct `type` for each

#### Contact
- [ ] Update email address
- [ ] Add phone number (optional)
- [ ] Update GitHub link
- [ ] Update LinkedIn link
- [ ] Update Twitter (optional)

#### SEO
- [ ] Update page title
- [ ] Update meta description
- [ ] Update keywords

---

### ☐ Step 3: Replace Resume (2 min)

- [ ] Export your resume as PDF
- [ ] Name it `resume.pdf`
- [ ] Place in `public/` folder
- [ ] Update filename in `portfolioData.js` if different

---

### ☐ Step 4: Test Everything (3 min)

- [ ] Scroll through all sections
- [ ] Click project links
- [ ] Test resume download
- [ ] Try dark/light mode toggle
- [ ] Check mobile view (resize browser)
- [ ] Verify all text updated

---

### ☐ Step 5: Build for Production (2 min)

```bash
npm run build
npm run preview
```

Check that everything works in production mode!

---

### ☐ Step 6: Deploy to Vercel (5 min)

#### Option A: Using GitHub (Recommended)
1. [ ] Create a GitHub account (if you don't have one)
2. [ ] Create a new repository
3. [ ] Push your code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin YOUR_REPO_URL
   git push -u origin main
   ```
4. [ ] Go to vercel.com
5. [ ] Sign in with GitHub
6. [ ] Click "New Project"
7. [ ] Import your repository
8. [ ] Click "Deploy"
9. [ ] Wait 2 minutes ⏰
10. [ ] Done! 🎉

#### Option B: Using Vercel CLI
1. [ ] Install Vercel CLI: `npm i -g vercel`
2. [ ] Run: `vercel`
3. [ ] Follow prompts
4. [ ] Done! 🎉

## 🆘 Troubleshooting

### Development server won't start
- [ ] Check Node.js version: `node --version` (need 18+)
- [ ] Delete `node_modules`
- [ ] Run `npm install` again

### Changes not showing
- [ ] Hard refresh: Ctrl + Shift + R (Windows) or Cmd + Shift + R (Mac)
- [ ] Clear browser cache
- [ ] Restart dev server

### Build fails
- [ ] Check for syntax errors in `portfolioData.js`
- [ ] Make sure all strings have closing quotes
- [ ] Check console for error messages
